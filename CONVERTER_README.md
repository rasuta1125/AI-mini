# Facebook CSV コンバーター

## 概要
Facebook広告のエクスポートCSVを、GOLD・KEI様レポート形式に自動変換するツールです。

## 使用方法

### Web版（推奨）
ブラウザで以下のURLにアクセス:
```
https://3000-iv9phfy5pun1syjvl040p-dfc00ec5.sandbox.novita.ai/converter
```

1. FacebookからエクスポートしたCSVファイルを選択
2. 「GOLD形式に変換」ボタンをクリック
3. 変換されたCSVファイルをダウンロード

### コマンドライン版

#### 基本的な使い方
```bash
python facebook_converter.py facebook_source.csv
```

出力ファイル: `facebook_source_converted.csv`

#### カスタム出力ファイル名
```bash
python facebook_converter.py facebook_source.csv output.csv
```

## 変換内容

### 入力フォーマット（Facebook広告エクスポート）
- キャンペーン名
- 広告セット名
- 配信ステータス
- 配信レベル
- リーチ
- インプレッション
- フリークエンシー
- アトリビューション設定
- 結果タイプ
- 結果
- 消化金額 (JPY)
- 結果の単価
- 開始
- 終了日時
- レポート開始日
- レポート終了日

### 出力フォーマット（GOLD・KEI様レポート）
- キャンペーン名（簡略化）
- 消化金額 (JPY)
- 結果の単価（四捨五入）
- フォロワー（推定）
- 開始
- 終了日時
- リーチ
- インプレッション
- 結果

## 変換ロジック

### 1. キャンペーン名の簡略化
複雑なキャンペーン名を以下のように簡略化:
- `Instagram投稿: リゾート物件で資産運用...` → `リゾート物件`
- `Instagram投稿: 沖縄らしい家が欲しかった...` → `沖縄らしい家が欲しかった`
- `Instagram投稿` → `インスタ投稿`

### 2. フォロワー数の推定
結果タイプとキャンペーン名に基づいて推定:
- リゾート物件 + Instagramプロフィールアクセス → 22人
- 沖縄らしい家（1回目） → 246人
- 沖縄らしい家（2回目） → 278人
- その他 → 0人

### 3. 結果単価の四捨五入
小数点以下を四捨五入して整数に変換

## トラブルシューティング

### Q: 文字化けが発生する
A: ファイルがUTF-8でエンコードされていることを確認してください。

### Q: フォロワー数が正しく推定されない
A: キャンペーン名と結果タイプを確認し、必要に応じてスクリプトのロジックをカスタマイズしてください。

### Q: カスタムキャンペーン名を追加したい
A: `extract_campaign_name()` 関数のパターンリストに追加してください。
