# AI広告分析ink

## プロジェクト概要
- **名前**: AI広告分析ink
- **目標**: Meta広告のCSVデータをアップロードするだけで、主要KPI（CTR/CPC/CPA/フォロー率）を自動算出し、グラフ可視化とAI改善コメントを表示するダッシュボード
- **機能**: CSV解析、KPI自動算出、クリエイティブランキング、グラフ可視化、AI分析コメント、レスポンシブデザイン

## 📊 現在完了している機能
✅ **CSVアップロード機能**
- Meta広告からエクスポートしたCSVファイルの自動認識
- 日本語列名対応（キャンペーン名、消化金額、結果、フォロワー、リーチ、インプレッション等）
- エラーハンドリングとファイル検証

✅ **KPI自動算出**
- CTR（クリック率）: 結果 ÷ インプレッション
- CPC（クリック単価）: 消化金額 ÷ 結果
- CPA（フォロワー単価）: 消化金額 ÷ フォロワー
- フォロー率: フォロワー ÷ リーチ
- 空値・ゼロ除算の安全な処理

✅ **グラフ可視化（Chart.js）**
- キャンペーン別CTR比較（棒グラフ）
- キャンペーン別CPA比較（棒グラフ）
- リーチ×フォロー率散布図
- インタラクティブな表示切り替え

✅ **クリエイティブランキング機能**
- 3つの指標による総合スコア算出（CTR 40%、フォロー率 30%、コスト効率 30%）
- トップ5キャンペーンのパフォーマンスランキング
- 採点基準のツールチップ表示
- アコーディオン式スコア内訳詳細表示
- 各指標の個別スコアと貢献度を100点満点で可視化

✅ **KPI説明機能**
- 各指標（CTR、CPC、CPA、フォロー率）の詳細解説モーダル
- 計算式、具体例、ベンチマーク、改善のヒントを表示
- 業界ベンチマークと最適化アプローチの提供

✅ **CSVエクスポート機能**
- 分析結果の詳細データをCSVファイルでダウンロード
- クリエイティブランキング、スコア内訳、貢献度を含む包括的なレポート
- サマリー統計（総消化金額、平均KPI等）も含む
- ファイル名に分析日付を自動付与（例：AI広告分析結果_2024-10-14.csv）

✅ **AI分析コメント機能**
- OpenAI API連携による自動分析
- 良い点・改善点・具体的提案の3区分
- 日本語での詳細なマーケティング分析

✅ **デザイン・UI/UX**
- 紫→青のグラデーション背景
- ガラスモーフィズム効果
- 5カード機能レイアウト（CSV/KPI/グラフ/ランキング/AI分析）
- クイックスタート（3ステップ）ガイド
- インタラクティブなツールチップとアコーディオンUI
- ホバーアニメーションとマイクロインタラクション
- レスポンシブデザイン（モバイル・タブレット対応）

## 📋 使い方ガイド
### ステップ1: CSVを用意
Meta Business Managerから広告データをCSVでエクスポートします。
必要な列：キャンペーン名、消化金額、結果、フォロワー、リーチ、インプレッション

### ステップ2: アップロードして分析
1. 「CSVファイルを選択」でファイルをアップロード
2. 「CSVを分析」ボタンをクリック
3. 自動でKPI算出、クリエイティブランキング、グラフが表示されます

### ステップ3: クリエイティブランキングを確認
1. パフォーマンス上位5キャンペーンのランキングを確認
2. 採点基準を知りたい場合は、情報アイコン（ⓘ）にホバー
3. 各キャンペーンをクリックしてスコア内訳詳細を展開

### ステップ4: KPI指標の詳細を学習
1. 「主要KPI」カードの「説明」ボタンをクリック
2. CTR、CPC、CPA、フォロー率の詳細解説を確認
3. ベンチマークと改善のヒントを参考にする

### ステップ5: AIコメントを確認
1. 「AI分析実行」ボタンをクリック
2. OpenAIによる分析結果を確認
3. 改善提案を次回の運用に活用

### ステップ6: 分析結果をCSVで保存
1. 「主要KPI」または「クリエイティブランキング」カードの「CSV出力」ボタンをクリック
2. 詳細なスコア内訳を含む分析レポートがCSVファイルでダウンロードされます
3. Excel等で開いてさらなる分析や報告資料作成に活用
